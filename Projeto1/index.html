O que é esse tal de HTML?

O HTML não é uma linguagem de programação, e sim uma linguagem de marcação, usada para definir a estrutura do seu conteúdo. O HTML consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes de um conteúdo para que ele apareça ou atue de determinada maneira. As tags anexas podem transformar uma palavra ou imagem num hiperlink, pode colocar palavras em itálico, pode aumentar ou diminuir a fonte e assim por diante.

Já a tag i, deixa o texto aninhado por ela em itálico

A tag u, deixa o texto aninhado como tachado abaixo do mesmo

A tag strong, deixa o texto aninhado em negrito, assim sendo usado para dar ênfase
Tambem temos as OL e UL. Que funcionam da seguinte forma, as OL, sendo listas ordenadas, como essa:
  1. Pão
  2. Café
  3. Suco

E temos as UL, que são o inverso, sendo listas, sem ordem. E elas são marcadas com esses bullets:
  • Dormir
  • Programar
  • Jogar

Já a tag a, ela te leva para links externo, servindo com âncora, nesse exemplo ao clicar será direcionado ao google

A tag sub irá deixar o texto aninhado por ela, com alinhamento abaixo.

Já a tag sup, faz o inverso, deixa o texto aninhado com alinhamento acima.

Já a tag blockquote é usada para citação, com um poema:
Lorem, ipsum dolor sit amet consectetur adipisicing elit. Saepe fuga at adipisci fugiat! Molestias quis sit delectus dicta ex? Doloribus sint culpa, vel ad ex quia cum commodi pariatur nisi.

A tag font, ela faz um pré formatação, sem necessidade de um css externo:
Texto em vermelho

A tag del deixa o texto aninhado com um tachado, bem no meio dele, para indicar uma possível errata!
